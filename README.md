# CRUD básico em spring boot

Esse projeto é feito de maneira simples para estudar o básico do spring boot, utilizando o spring data jpa para persistência de dados, H2 database para uma visualização mais rápida e prática do resultado do trabalho e o spring web para criar uma API REST.

O projeto não tem a intenção de demonstrar complexidade, apenas a visualização do básico do framework java.

O projeto consiste em uma entidade representando um item de um desktop e uma entidade associativa representando o tipo do item (uma pasta, aplicativo, etc...).
